# Final Diff — V13 vs V15

## V13
V13 already had:
- real sending profiles inside the website
- SMTP and manual Graph token support
- AI Outreach Agent
- campaigns, sequences, replies, suppression

## V15
V15 adds the product layer that makes it stronger for company rollout and later resale:

### Productization upgrades
- Product Settings page for white-label reuse
- Export Engine workspace instead of only CRM + Outreach
- Product catalog and market notes stored inside the app
- Unified search covers export data too

### Email / Microsoft upgrades
- Microsoft OAuth start + callback routes
- linked Microsoft account support on sending profiles
- Inbox Sync screen for reply harvesting from Microsoft Graph
- reply sync now creates a follow-up task automatically
- Graph token refresh path added for linked accounts

### Business / UX upgrades
- clearer navigation for Export Engine
- search visibility for products and markets
- AI Export Agent that drafts market/product-specific export messaging

## Simple business summary
### Old build
A strong internal CRM + outreach tool.

### New build
A clearer product:
- CRM
- Outreach Engine
- Export Engine
- white-label product settings
- Microsoft 365 connection path
