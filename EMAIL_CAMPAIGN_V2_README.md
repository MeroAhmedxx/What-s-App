# Email Campaign V2 for Altahhan CRM

This package adds a safer Campaign V2 module on top of the working CRM.

Included now:
- sending profiles per user
- linked mailbox placeholders per user
- personal/team email templates
- campaign builder with selected leads
- send-now processing using existing SMTP or Outlook Desktop mode
- send attempts log
- open/click tracking via existing /track/open and /track/click
- manual reply marking for workflow follow-up

Working send modes now:
- smtp
- outlook_desktop

Planned later:
- microsoft_graph
- google_api
