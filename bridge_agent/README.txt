1) Install Python on the employee Windows PC.
2) Install Classic Outlook and sign in.
3) pip install -r requirements.txt
4) Edit run_bridge.bat with BASE_URL and CRM_DEVICE_TOKEN
5) Run run_bridge.bat and keep Outlook open.


Mobile note:
- Mobile users can review campaigns and bridge status from the web app.
- Only desktop Windows devices with Classic Outlook can execute email sending via the agent.
